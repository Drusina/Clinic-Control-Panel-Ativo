## 🐛 PROBLEMAS COMUNS E SOLUÇÕES

### BrasilAPI retorna CORS error
- **Causa:** BrasilAPI aceita CORS de qualquer origem. Se falhar, provavelmente é CNPJ inválido.
- **Solução:** validar o CNPJ antes com dígito verificador, ou mover a chamada para Edge Function Next.js (`/api/cnpj/[cnpj]`)

### Autentique não cria documento
- **Causa:** Token inválido ou PDF com problema
- **Solução:** testar o token em [studio.apollographql.com](https://studio.apollographql.com) com o endpoint `https://api.autentique.com.br/v2/graphql`, header `Authorization: Bearer {token}`

### RLS bloqueia queries
- **Causa:** Usuário não tem vínculo em `clinic_users`
- **Solução:** verificar a tabela `clinic_users` e a função `user_clinic_ids()`. Super admin deve ter `profiles.is_super_admin = true`.

### Claude API retorna 429
- **Causa:** rate limit do plano free
- **Solução:** upgrade para plano build ou implementar retry exponencial

### shadcn/ui components não aparecem
- **Causa:** falta importar no projeto
- **Solução:** `npx shadcn-ui@latest add button card dialog input select table tabs`
